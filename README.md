Transform your photos into stunning AI-generated clipart styles.

Features

5 AI art styles: Cartoon, Flat Art, Anime, Pixel Art, Sketch
Before/After slider to compare original and generated
Save and share generated images
Beautiful gradient UI with smooth animations
Tech Stack

React Native (Expo)
Stability AI (image-to-image transformation)
Express.js backend (Render.com)
expo-linear-gradient, expo-image-picker, expo-media-library
Setup

Clone the repo
Run npm install
Run npx expo start
Scan QR code with Expo Go
Backend

Repo: https://github.com/Dilashanegi/ClipartBackend
Deployed on Render.com
Uses Stability AI API for image transformation
